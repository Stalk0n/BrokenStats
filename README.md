# Broken Stats

**Broken Stats** to aplikacja desktopowa dla graczy **Broken Ranks**, umożliwiająca zbieranie, przechowywanie i analizowanie danych związanych z rozgrywką.

## Funkcje

* Rejestrowanie statystyk z walk:

  * zdobyte doświadczenie (EXP),
  * zdobyte złoto,
  * zdobyte przedmioty (drop).

* Śledzenie informacji o respawnach:

  * championów,
  * elit.

* Zapisywanie i przeglądanie logów czatu.

* Wbudowane kalkulatory wspomagające rozgrywkę.

* Analiza zgromadzonych danych i historii aktywności.

## Zastrzeżenie

Broken Stats jest projektem stworzonym przez społeczność graczy i nie jest powiązany z twórcami gry Broken Ranks.
